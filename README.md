# 宠物管理系统 java192

## 项目概述

宠物管理系统是一款集宠物信息管理、领养、救助等功能于一体的后台管理系统。系统采用前后端分离的设计，基于Java和SpringBoot技术，前端使用Vue框架。

## 技术栈

- 后端：Java、SpringBoot
- 数据库：Mysql
- 前端：Vue
- 开发环境：IDEA、JDK1.8

## 系统角色

- 前台用户：可以查看宠物信息、领养宠物、发布救助信息等。
- 后台管理员：负责管理宠物信息、用户领养申请、救助站信息等。

## 运行环境

- 系统环境：Windows、Mac
